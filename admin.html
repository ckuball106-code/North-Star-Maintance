One-Time Setup

You need a GitHub token so photos can be saved to your website. This only needs to be done once on this device.

How to get your token:

  1. Go to github.com/settings/tokens
  2. Tap "Generate new token"
  3. Name it Gallery Admin
  4. Under Repository access → Only select repositories → pick North-Star-Maintance
  5. Under Permissions → Contents → set to Read and write
  6. Tap Generate token, then copy and paste it below
